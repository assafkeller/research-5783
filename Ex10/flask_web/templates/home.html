hyper network that learns to generate the filter weights of an image restoration network conditionally based on the required restoration level, given as an input parameter.
To ensure an efficient representation at inference time, we constrain the kernels corresponding to the different levels to be identical up to a scaling factor and learn only the weights basis vectors.
